    WordPress 是什麼？5分鐘快速帶你瞭解 WordPress 的好用之處

    2021年9月 大新

    WordPress 採用 GPL v2 的開源協議，是一款開放式的內容管理系統，因此全世界每個人都能夠免費的修改與編輯 WordPress 這套系統，就像是 Linux 一樣。 WordPress 作為一套內容管理系統，意味著你能夠輕鬆的透過它來管理網站、發表文章…等等功能，最重要的是，你不需要自己造輪子、不需要會寫程式，也能夠輕鬆地利用 WordPress 來快速建立網站。 如果你是程式開發人員，那麼 WordPress 的自由度與靈活度，也能好好的讓你大展身手，透過開發外掛，你能夠在 WordPress 網站當中實現各種想要的特殊功能，只要你寫的出來，就絕對能夠應用與實現在 WordPress 網站當中。

    一開始，WordPress 主要是用來作為部落格的一套內容管理系統，其功能非常簡單，且沒有太多的主題與外掛能夠使用。 但如今，WordPress 經過許多人的推廣與各路開發者的努力，現在的 WordPress 擁有非常健全的社區與開發環境，同時也累積了大量的主題與外掛，整個 WordPress 的生態系統可謂是相當完整。 因此，不管你現在想要建立哪一種類型的網站，都能夠使用 WordPress 來幫你完成。

    通常當我們稱「WordPress」時，一般指的是 WordPress.org 而非 WordPress.com。 如果你想要完全掌握自己的網站，那麼 WordPress.org 是你最佳的選擇； 如果你是初學者，只想要一個部落格平台來寫作，那麼 WordPress.com 是你最佳的選擇。 當你使用 WordPress.org 來建立網站時，你會需要自行購買網域名稱與自行購買虛擬主機。
